Side by side comparison between Ovi and Javis, Universe-1 using Verse-Bench set 1 with total 205 evaluation prompts with ref images (random 25 for display here)

Protocol & Viewing Guide

  1. Modality matching for fair comparison on both Javis and Universe-1. We evaluate text-to-video (T2V) baselines by comparing javis (T2V) with ovi_t2v. We evaluate image-to-video (I2V) baselines by comparing universe_i2v (I2V) with ovi_i2v. Each comparison matches the single modality supported by the respective baseline.
  2. Organization of examples. Each four-video group corresponds to a single prompt identifier. For each identifier, two rows are shown:
    Row 1: javis vs ovi_t2v (T2V) Row 2: universe_i2v vs ovi_i2v (I2V)
  3. Prompt formatting. For I2V comparisons, we use the original JSON prompt format aligned with the universe_i2v model requirements. For T2V comparisons, we merge the video and audio fields into a single text prompt as required by the javis interface. The same semantic content is used across systems within each modality.
  4. Qualitative evaluation criteria. Labelers are presented with a random pair of videos each time, and are asked the six questions below in terms of evaluation criteria. Labelers are asked to select one of the three choices as answer to each question - Left is Better, Right is Better or Not Sure.
    1. (Video Quality) Which example gives better overall video quality and structural correctness of human and objects if presented,
    2. (Audio Quality) Which example gives better speech, music, and sound generation quality,
    3. (Audio-Visual Sync) Which example gives better Lip-sync correctness if speech is involved in the prompt,
    4. (Video Quality) Which example gives better temporal consistency across the entire video,
    5. (Video Quality) Which example gives larger motion range of the generated video,
    6. (video Quality) Which example gives better Prompt-following fidelity.
  5. Summary statement. Blind voting results from reviewers suggest that ovi performs competitively or favorably on the above criteria with a large margin in both T2V and I2V settings to both Javis and Universe-1 as written in the paper
Each group is labeled with its JSON id (e.g., 00000.json) so reviewers can cross-reference the exact prompt used for generation. This page shows a randomly selected subset (up to 25 groups) due to size limits.
